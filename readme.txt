== Description ==

基于用户自定义顶贴内容（比如顶、赞、顶一个，感谢分享等），无需要人工干预自动顶贴，让平台人气更加活跃。

## 插件功能

* 支持发完文章或更新文章后，文章后自动实现顶贴，自动活跃平台氛围
* 支持顶贴用户名
* 自定义顶贴内容，自动评论，支持Emoji表情
比如
👍顶得飞起！
👏说得对！
💯赞一个！
等，分割符号英文回车换行
* 支持随机间隔时间评论,时间秒数，比如 30~600 30秒到600秒之间随机评论，更加真实
